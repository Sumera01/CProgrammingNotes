Function Arguments and Return Values

1. Function with Arguments and Return Value

cCopy code
#include <stdio.h>
// Function declaration (prototype)
int add(int a, int b);

int main() {
    int num1 = 5, num2 = 10;
    int sum;

    // Function call
    sum = add(num1, num2);

    // Print the result
    printf("Sum: %d\n", sum);

    return 0;
}

// Function definition
int add(int a, int b) 
{
    return a + b;
}

Explanation:

  • Function Declaration: int add(int a, int b); declares a function add that takes two integer arguments (a and b) and returns an integer (int).
  • Function Call: sum = add(num1, num2); calls the add function with num1 and num2 as arguments.
  • Function Definition: int add(int a, int b) { return a + b; } defines the add function, which calculates the sum of a and b and returns the result.

2. Function with Arguments but No Return Value

cCopy code
#include <stdio.h>// Function declaration (prototype)
void greet(char name[]);

int main() {
    char userName[] = "Alice";

    // Function call
    greet(userName);

    return 0;
}

// Function definition
void greet(char name[]) {
    printf("Hello, %s!\n", name);
}

Explanation:

  • Function Declaration: void greet(char name[]); declares a function greet that takes a character array (name) as an argument and returns nothing (void).
  • Function Call: greet(userName); calls the greet function with userName as an argument.
  • Function Definition: void greet(char name[]) { printf("Hello, %s!\n", name); } defines the greet function, which prints a greeting message using the provided name.

3. Function without Arguments but with Return Value

cCopy code
#include <stdio.h>// Function declaration (prototype)
int getRandomNumber();

int main() {
    int randomNumber;

    // Function call
    randomNumber = getRandomNumber();

    // Print the result
    printf("Random Number: %d\n", randomNumber);

    return 0;
}

// Function definition
int getRandomNumber() {
    return rand() % 100; // Return a random number between 0 and 99
}

Explanation:

  • Function Declaration: int getRandomNumber(); declares a function getRandomNumber that takes no arguments and returns an integer (int).
  • Function Call: randomNumber = getRandomNumber(); calls the getRandomNumber function.
  • Function Definition: int getRandomNumber() { return rand() % 100; } defines the getRandomNumber function, which generates and returns a random number between 0 and 99 using the rand() function.

4. Function without Arguments and Return Value

cCopy code
#include <stdio.h>// Function declaration (prototype)
void sayHello();

int main() {
    // Function call
    sayHello();

    return 0;
}

// Function definition
void sayHello() {
    printf("Hello, World!\n");
}

Explanation:

  • Function Declaration: void sayHello(); declares a function sayHello that takes no arguments (void) and returns nothing (void).
  • Function Call: sayHello(); calls the sayHello function.
  • Function Definition: void sayHello() { printf("Hello, World!\n"); } defines the sayHello function, which prints "Hello, World!" when called.

Conclusion

Understanding different types of functions based on their arguments and return values is crucial in C programming. Functions allow you to modularize code, improve reusability, and enhance program structure. Mastering how to declare, define, and use functions with various argument and return value combinations is essential for developing efficient and maintainable programs.
