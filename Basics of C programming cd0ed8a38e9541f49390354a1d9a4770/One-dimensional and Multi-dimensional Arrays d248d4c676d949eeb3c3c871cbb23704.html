One-dimensional and Multi-dimensional Arrays

One-dimensional Arrays

One-dimensional arrays in C are collections of elements stored in a sequential order. They are useful for storing and manipulating linear data structures, such as lists of numbers, characters, or strings.

Declaration and Initialization of One-dimensional Arrays

Syntax:


dataType arrayName[arraySize];

Example:


int marks[5]; // Declaration of an integer array to store 5 marks
float prices[10]; // Declaration of a floating-point array to store prices of 10 items
char vowels[5]; // Declaration of a character array to store 5 vowels

Arrays can be initialized at the time of declaration by providing initial values enclosed in curly braces {}.

Example:


int numbers[5] = {10, 20, 30, 40, 50}; // Initialization of an integer array
char name[6] = {'J', 'o', 'h', 'n', ' ', '\0'}; // Initialization of a character array (string)

Accessing Elements in One-dimensional Arrays

Elements in a one-dimensional array are accessed using index values starting from 0 up to arraySize - 1.

Example:


int marks[5] = {85, 90, 75, 80, 95};
printf("First mark: %d\n", marks[0]); // Accessing first element
printf("Third mark: %d\n", marks[2]); // Accessing third element

Multi-dimensional Arrays

Multi-dimensional arrays in C are arrays of arrays. They are useful for storing and manipulating tabular data, matrices, and other complex data structures that require more than one index to access elements.

Declaration and Initialization of Multi-dimensional Arrays

Syntax:


dataType arrayName[rowSize][colSize];

Example:


int matrix[3][3]; // Declaration of a 3x3 integer matrix
float grades[5][4]; // Declaration of a 5x4 floating-point grades matrix
char board[8][8]; // Declaration of an 8x8 character chess board

Multi-dimensional arrays can also be initialized at the time of declaration by providing initial values enclosed in nested curly braces {}.

Example:


int identityMatrix[3][3] = {{1, 0, 0}, {0, 1, 0}, {0, 0, 1}}; // Initialization of a 3x3 identity matrix

Accessing Elements in Multi-dimensional Arrays

Elements in a multi-dimensional array are accessed using multiple index values corresponding to each dimension.

Example:


int matrix[2][3] = {{1, 2, 3}, {4, 5, 6}};
printf("Element at matrix[0][1]: %d\n", matrix[0][1]); // Accessing element at row 0, column 1
printf("Element at matrix[1][2]: %d\n", matrix[1][2]); // Accessing element at row 1, column 2

Example Program: Sum of Matrix Elements


#include <stdio.h>#define ROWS 3
#define COLS 3

int main() {
    int matrix[ROWS][COLS] = {
        {1, 2, 3},
        {4, 5, 6},
        {7, 8, 9}
    };
    int sum = 0;
    int i, j;

    // Calculate sum of matrix elements
    for (i = 0; i < ROWS; i++) {
        for (j = 0; j < COLS; j++) {
            sum += matrix[i][j];
        }
    }

    printf("Sum of matrix elements: %d\n", sum);

    return 0;
}

Explanation: This program calculates the sum of elements in a 3x3 matrix (matrix) using nested for loops to iterate through each element and accumulate the sum.

Conclusion

One-dimensional and multi-dimensional arrays in C programming provide efficient ways to store and manipulate collections of data. Understanding how to declare, initialize, access elements, and use loops to process arrays is essential for developing applications that require handling structured data effectively.
