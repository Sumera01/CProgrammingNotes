Structure Declaration and Initialization

Declaring Structures

  • Structure Declaration: A structure must be declared before it can be used. The declaration defines a new data type that can be used to create variables.

    Syntax:

    cCopy code
struct StructureName {
    dataType member1;
    dataType member2;
    // more members
};

    Example:

    cCopy code
struct Student {
    int rollNumber;
    char name[50];
    float marks;
};

    In this example, Student is a structure with three members: rollNumber, name, and marks.

Declaring Structure Variables

  • Once a structure is defined, you can declare variables of that type.

    Syntax:

    cCopy code
struct StructureName variableName;

    Example:

    cCopy code
struct Student student1;

Initializing Structures

  • Initialization at Declaration: You can initialize a structure variable at the time of declaration.

    Syntax:

    cCopy code
struct StructureName variableName = {value1, value2, ...};

    Example:

    cCopy code
struct Student student1 = {101, "John Doe", 95.5};
  • Individual Member Initialization: You can also initialize individual members of a structure after declaration.

    Example:

    cCopy code
struct Student student2;
student2.rollNumber = 102;
strcpy(student2.name, "Jane Doe");
student2.marks = 89.0;

Example: Structure Declaration and Initialization

cCopy code
#include <stdio.h>#include <string.h>// Defining the Student structure
struct Student {
    int rollNumber;
    char name[50];
    float marks;
};

int main() {
    // Initializing structure at declaration
    struct Student student1 = {101, "John Doe", 95.5};

    // Declaring and then initializing structure
    struct Student student2;
    student2.rollNumber = 102;
    strcpy(student2.name, "Jane Doe");
    student2.marks = 89.0;

    // Printing structure members
    printf("Student 1 Roll Number: %d\n", student1.rollNumber);
    printf("Student 1 Name: %s\n", student1.name);
    printf("Student 1 Marks: %.2f\n", student1.marks);

    printf("Student 2 Roll Number: %d\n", student2.rollNumber);
    printf("Student 2 Name: %s\n", student2.name);
    printf("Student 2 Marks: %.2f\n", student2.marks);

    return 0;
}

Explanation:

  • Structure Definition: The struct Student is defined with three members: rollNumber, name, and marks.
  • Initialization at Declaration: struct Student student1 = {101, "John Doe", 95.5}; initializes student1 at the time of declaration.
  • Separate Initialization: struct Student student2; declares student2, and then its members are initialized individually.
  • Printing Members: The values of the members of both student1 and student2 are accessed and printed.

Conclusion

Declaring and initializing structures properly is crucial for managing complex data in C. This helps in grouping related information and makes it easier to handle records.
